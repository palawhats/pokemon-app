--Pokemon App (Pokedex)--

เว็บสารานุกรมโปเกมอน พัฒนาด้วยเฟรมเวิร์ก Next.js
ตกแต่งด้วย Material UI (MUI v6)

เทคโนโลยีที่เลือกใช้
- Framework: [Next.js 14/15 (App Router)](https://nextjs.org/)
- Language: TypeScript
- UI Library: [Material UI (MUI v6)](https://mui.com/)
- Styling: CSS Modules / MUI `sx` Prop
